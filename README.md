# # Student Management System (Java)

This is a simple console-based Java project.
It performs basic operations like add, view, update, and delete student records.
Built using Core Java concepts.

## Features
- Add new student
- View all students
- Update student details
- Delete student
- Console-based menu

## Technologies Used
- Java
- OOP concepts
- Scanner class

## How to Run
1. Download the repository
2. Open the project in any Java IDE or online compiler
3. Run the Main.java file
4. Follow the menu shown in console
